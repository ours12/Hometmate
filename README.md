# 홈트메이트 - WebRTC 화상 홈트레이닝 플랫폼
2022년 3월 03일 ~ 2022년 2월 31일 (4주)
## 프로젝트 소개
WebRTC 를 이용한 화상 홈트레이닝 플랫폼
WebSocket을 이용한 채팅 및 유튜브 실시간 공유 & 재생
Story 기능을 통한 유저들간의 식단 및 운동 공유
